oregon wild

This project was completed as part of my CSS course at code school. The focus with this assignment was to practice creating a responsive site that looked great regardless of what device the user was viewing it from. This meant implementing a module design that would dynamically resize with the inclusion of media queries into the styling. The assignment was centered solely on the visual presentation of the landing page, so I only used html and SASS to build it out with placeholder links and no JavaScript.

Responsive design

From the beginning of my web development education, I have been taught to design and code starting with mobile first and scale up from there - this project was no different. With the impact of this design coming largely from its beautiful photos, I ensured they remained front and center in every screen size, taking up the majority of the viewport.
