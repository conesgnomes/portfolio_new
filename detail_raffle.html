Sq1 Raffle Site

One project I was asked to work on while interning at Sq1, a creative agency in Portland, Oregon, was redesigning the UX and UI for the company's internal raffle system. The system was already being used internally, but there were a number of changes and imrovements that had been requested by users since the system went live. I took into account all of these requests to design a more beautiful and streamlined tool.

Inspiration is found everywhere

The look we landed on for the branding of this project was inspired by the book Around the World in Eighty Days by Jules Verne, the name itself a play on Passepartout, a main character in the story. We brought the theme to life largely through the imagery in our landing page with the inclusion of giant blimps, hot air balloons, and famous monuments from around the world.

JavaScript and style fundamentals

Looking back on this project what still impresses me about our development process was our implementation of JavaScript and CSS. As a class, we had only eight days worth of JavaScript instruction and four of CSS. With this project we dug deep into our basic knowledge and pushed the limits of what we were capable of. The outcome was a beautiful and dynamic user interface and a user experience that is simple and easy to use.
