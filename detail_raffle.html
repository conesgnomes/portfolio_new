Sq1 Raffle Site

One project I was asked to work on while interning at Sq1, a creative agency in Portland, Oregon, was redesigning the UX and UI for the company's raffle system. The system was already being used internally, but there were a number of changes and improvements that had been requested by users since the system went live. I took into account all of these requests to design a more beautiful and streamlined tool.

Updated user experience and interface

In redesigning the user interface for Sq1's raffle app, I wanted to maintain the original aesthetic, but updated its overall feel, while making the UX more straightforward. I did this by including easy-to-read buttons on each raffle, making clear how to edit, delete, and see participants in the raffle. I also moved the 'New Raffle' button down into the main viewing area of the screen and made it large with descriptive text. Additionally I incorporated slight gradients, large, simple iconography, and tasteful integration of drop shadows to modernize the look and feel. For comparison, see the old interface on top.

A more streamlined user flow

The original raffle app that I was redesigning often left the user with questions as they navigated through the interface: how do I register for a raffle? How do I create a new raffle? Who has already registered for a particular raffle? With my design I help users answer these questions through straightforward button placement and strong visual cues, as well as a more predictable flow through application.
